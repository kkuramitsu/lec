    配列（NumPy)#

    データサイエンスや機械学習の基礎となるリストとNumPyの使い方を学びます。

    準備#

    Pythonで拡張機能を用いたプログラミングをするときは、まずモジュールをインポートして使います。

    Numpyとは#

    Numpyは、科学計算やデータサイエンス、機械学習で最もよく使われる基本的なライブラリです。 多次元配列を効率よく処理することができます。

    Numpy モジュールをnpという名前でインポートします。

    import numpy as np
import numpy.random as random
%precision 3

    リストと配列#

    リストは、複数のデータを列として扱うデータ構造です。 次のように[ ] で囲んでリストを作ります。

    リストの例

    リストの例
    data = [1, 2, 3, 4, 5, 6] 
print(data)

    NumPy配列は、科学技術計算のため開発されたリストの高効率＆高速版です。

    NumPyの配列: リストの高速版

    NumPy配列
    data = np.array([1, 2, 3, 4, 5, 6])
print(data)

    データサイエンスや機械学習では配列を使うことが多いので、 NumPyの配列の使い方を学びます。

    リスト/配列の基本操作#

    リストや配列は、インデックス(添字)によって内部の値を取り出すことができます。

    Example

    例題（配列）

    次の配列aに対し、次の値を求める操作を書いてみよう

    入力例：

    a = np.array([1, 2, 3, 4, 5, 6, 7, 8])
    1. aの最初の要素
    2. aの要素数を得る
    3. aの先頭と末尾の和
    4. aの先頭を取り除いた配列
    5. aを逆順にした配列
    6. aの総和
    7. aの平均値

    最初の要素

    print(a[0])

    Warning

    個数える

    プログラミング(Python)では、原則、「0からまで」のように0から数えます。

    要素数を得る

    len(a)

    aの先頭と末尾の和

    a[0]+a[len(a)-1]

    添字を-nにすると、末尾から位置を指定できます。

    a[0]+a[-1]

    aの先頭を取り除いた配列

    a[1:]

    aを逆順にした配列

    a[::-1]

    aの総和

    a.sum()

    aの平均値

    a.mean()

    まとめ#

    数列(数学) リスト(Python) 意味
    len(a) 個数
    a[0] 先頭の値
    a[1]
    a[i] i番目の値
    a[len(a)-2]もしくはa[-2]
    a[len(a)-1]もしくはa[-1] 末尾の値

    配列の演算#

    Numpyの配列は、科学技術計算用に開発されたため、データは演算はベクトルとして解釈されます。

    a = np.array([1, 2, 3, 4])
b = np.array([1, 0, 0, 1])

    配列の和

    a + b

    配列の差

    a + b

    配列のスカラー積

    a * 2

    内積

    a * b

    次元の変更

    a.reshape(2,2)

    数列の生成#

    最後に、データサイエンスで用いる便利な数列の生成を紹介しておきます。

    等差数列 (つまり、range(x)のNumPy版)

    np.arange(10)  # 0から10未満までの整数列

    np.arange(1, 101, 5)  # 1から100までの5刻みの整数列

    区間を指定した数列

    こちらは、要素の数をズバリ指定します。

    np.linspace(-10, 10, 20)  # 区間[-10, 10] 要素数20の数列

    同じ値の数列

    20個の0からなる数列

    np.zeros(20)  # 0が20個

    np.ones(15)  # 1が15個

    乱数列

    np.random.rand(100)   # 0.0から1.0までの乱数列(100個)

    np.random.randint(1, 6, 100) #1から6までの乱数列(100個)

    グラフの描画#

    数列は、グラフ化すると、視覚的にみやすく、特徴が理解しやすくなります。

    例えば、x, yの数列を表示してみても、xとyの関係性はわかりません。

    x = np.linspace(-10, 10, 100)
y = np.sin(x)
print(x)
print(y)

    Matplotlib でグラフに表示してみると、視覚的にわかりやすくなります。

    x = np.linspace(-10, 10, 100)
y = np.sin(x)
plt.plot(x, y)
plt.grid(True)

    Warning

    ユニバーサル関数: np.sin()

    np.sin(x)は、NumPy用の関数です。配列xの各要素ごとにを計算します。

    グラフの書き方#

    の例から、グラフの書き方を学びましょう

    2. y 軸の数列を計算する

    4. plt.plot(x, y) でグラフにプロット
    x = np.linspace(-10, 10, 100)

plt.figure(figsize=(10, 2)) #グラフの大きさを指定
plt.plot(x, y, label='sin(x)')

    色々なグラフ#

    Matplotlibは色々なグラフを描画することができます。少し例をみながら書き方を学びましょう。

    散布図 (plt.scatter(x,y))#

    散布図は、2つのデータの組み合わせに対して、xy座標上にプロットしたグラフです。散布図を書くと、ふたつのデータの関連性が見えてきます。

    一様乱数列 X, Yの散布図

    x = np.random.rand(100)
y = np.random.rand(100)
plt.figure(figsize=(4,4))
plt.scatter(x, y, marker='x')

    色分けして表示する例

    モンテカルロ法で、円周率を求めるとき、半径１の円の中にあるかないかで色分けしてみます。

    # モンテカルロ法
x = np.random.rand(200)
y = np.random.rand(200)
d = np.hypot(x, y)   # 原点から(x,y)の距離

# 原点からの距離で分ける
x_inside = x[d<1.0]  
y_inside = y[d<1.0]
x_outside = x[d>=1.0]
y_outside = y[d>=1.0]

plt.figure(figsize=(5,5))
plt.scatter(x_inside,y_inside, c='pink')　# ピンク色
plt.scatter(x_outside,y_outside, c='cyan') # シアン色

    Warning

    配列のフィルタ

    フィルタは、条件にマッチした値だけ残す操作です。少し特殊ですが配列[条件式]のように書きます。

a[a % 2 == 0] # 偶数かどうか

    ヒストグラム#

    ヒストグラムは、縦軸に度数、横軸に階級をとり、データの分布状況を視覚的に認識するために定番のグラフです。

    サイコロを60回ふったときの各目の出現

    x = np.random.randint(1, 7, 60) #1以上7未満の60個の乱数
print(x)
plt.hist(x, bins=6, rwidth=0.8) 
plt.axhline(y = 10, color='red', linestyle='--') #期待値に赤点戦を引く

    Subplot 複数の図を表示する#

    Subplotの機能を使えば、複数の図をみやすいように一枚にまとめて描画することができます。

    # https://matplotlib.org/tutorials/introductory/pyplot.html より
def f(t):
    return np.exp(-t) * np.cos(2*np.pi*t)

t1 = np.arange(0.0, 5.0, 0.1)
t2 = np.arange(0.0, 5.0, 0.02)

plt.figure(1)
plt.subplot(2,1,1)  # 1枚目の位置
plt.plot(t1, f(t1), 'bo', t2, f(t2), 'k')

plt.subplot(2,1,2)  # 2枚目の位置
plt.plot(t2, np.cos(2*np.pi*t2), 'r--')
plt.show()

    Warning

    subplot の理解するためには

    グラフを1枚追加して、plt.subplot(3,1,3)で描画してみよう。もしくは、subplot()のパラメータを変更してどうなるか変わるか調べてみよう。
