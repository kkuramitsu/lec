    回帰分析: 数値を予測する#

    今回は、回帰分析に取り組み、統計手法を応用して数値を予測する手法 を学んでいきます。 ハンズオン演習のデータセットは、前回までの同じボストン住宅価格データセットを使います。

    準備#

    まず、前回の演習内容を思い出して、ボストン住宅価格データセットをdfに読み込んでください。

    ハンズオン演習を始めるため、Python モジュールとデータセットを読み込んでおきましょう。

    1. データ分析用のモジュールを一式インポートする
    2. ボストン住宅価格データセットをダウンロードする
    3. バイアスを取り除いて、データフレームを用意する

    ちゃんと準備できましたか？

    線形回帰#

    前回、散布図と相関係数の求め方を学びました。 最後に、強い相関がある散布図にもっともらしい直線()を引く方法を紹介しました。

    最小二乗法の原理はさておき、 もっともらしい直線に近似する ところは、そういうものかなと理解できたと思います。

    実は、データの散らばりを線形の数式()に近似するのは、 線形回帰(Linear Regression) と呼ばれます。

    説明変数と目的変数#

    回帰分析やこれ以降の機械学習では、 やのように関係を表す式をモデル、 を説明変数、を目的変数と呼びます。

    • 説明変数: モデルの特徴を説明するための変数
    • 目的変数: モデルの対象となる変数

    データ列の中からひとつの説明変数を選んで、回忌分析を行うことを 単回帰分析 と呼びます。

    説明変数と目的変数を抽出
    X = df[['部屋数']]  # ひとつ選ぶ
y = df['住宅価格']
print(X.shape) #データ数と次元

    (506, 1)

    説明変数は、２つ以上選ぶことができます。 ２つ以上の線形回帰は、重回帰分析 と呼ばれます。

    説明変数を二つ抽出する
    X = df[['部屋数', '築年']]
y = df['住宅価格']
print(X.shape) #データ数と次元

    (506, 2)

    説明変数はモデルの特徴をうまく表す変数になります。 もちろん、３つやそれ以上の説明変数を選んでも構いません。

    説明変数の次元を確認
    X = df[['部屋数', '築年', '犯罪率']]
print(X.shape) #データ数と次元

    (512, 3)

    次元(dimension)

    ２次元以上の説明変数は、ベクトルとして計算されます。 だから、説明変数の数を 次元(dimension) と呼びます。

    ようこそ、初めての機械学習の世界へ#

    回帰分析は、回帰モデルを作ることです。 回帰モデルは、説明変数と目的変数から算出するので、 説明変数と目的変数を教師データとして機械学習することになります。

    回帰モデルは、 sklearn.linear_model.LinearRegressionを用います。 このように書いたとき、 sklearn.linear_modelがモジュール名、LinearRegressionがモデルを表現するクラス名になり、 次のようにインポートします。

    sklearn.linear_model.LinearRegressionのインポート
    from sklearn.linear_model import LinearRegression

    新しいモデルは、LinearRegressionから初期化します。

    初期化
    model = LinearRegression()

    これで、modelは、いわゆる学習前のモデルが用意されたことになります。 このmodelに対して、fit()メソッドで、 説明変数Xと目的変数yを与えると、モデルを学習します。

    まずは、説明変数を一つ選んで、単回帰モデルを作ってみましょう。

    単回帰モデルの学習
    X = df[['部屋数']]
y = df['住宅価格']

model = LinearRegression()
model.fit(X, y)

    機械学習とは

    機械学習とは、言い換えるとデータを与えることで、そのデータに適合したモデルを得ることです。

    適合された回帰モデルから次のように係数(a)と切片(b)を得ることができます。

    回帰モデル(係数と切片）の確認
    print('a=', model.coef_, ', b=', model.intercept_)

    a= [9.10210898] , b= -34.67062077643857

    つまり、数式で表すと、次のような直線になります。

    回帰モデルと予測#

    回帰モデルでは、一旦、モデルが構築されると、 モデルを用いて、目的変数を予想できるようになります。

    部屋数4の価格は？
    model.predict([(4.00,)])

    array([19.94203311])

    次のように、y_predを求めると、 説明変数Xの全ての値から予測された住宅価格を得ることができます。

    回帰モデルから説明変数を全て予測
    y_pred = model.predict(X)
print(y_pred)
    実行結果(抜粋)
    [25.17574577 23.77402099 30.72803225 29.02593787 30.38215211 ...]

    線形回帰と数値予測

    実は、回帰分析でもっともらしい直線を近似することは、 説明変数から目的変数を予測するモデルを作ることになります。 これは、機械学習やAIで正に目指していることです。

    データ列を眺めていてもよくわかりませんよね。 グラフを描画して可視化してみましょう。

    回帰モデルはどの程度正確なのか？#

    回帰モデルによる予測は、どの程度、正確なのでしょうか？

    学習された回帰モデルを使って、価格を予測して、実際の住宅価格（正解）と比較してみましょう。

    y_pred = model.predict(X)
pd.DataFrame({'実測': df['住宅価格'], '予測':y_pred}).head()

    やはり、データ列を比較するだけではわかりにくいですね。

    散布図などを使って視覚化してみます。

    散布図で視覚化
    plt.scatter(y, y_pred, color = 'red', alpha=0.3) 
plt.xlabel('実測')
plt.ylabel('予測')
plt.ylim(0, 50)

    なんか高価格帯()がうまく予測できていない気がします。

    残差

    残差とは、実測値と予測値の差です。 回帰モデルは、残差が正規分布になっているのが望ましいと言われています。

    残差のヒストグラム
    plt.hist(y-y_pred, bins=50)
plt.show()

    定量的な評価尺度#

    回帰モデルの正確さは、実測値と予測値の散布図や残差ヒストグラムで視覚的に確認できます。 しかし、定量的に評価できる尺度があった方が便利です。 そのような回帰モデルを定量的に評価する尺度として、MSE と決定係数 が使われます。

    MSE#

    平均二乗誤差 (Mean Squared Error) とは、実際の値と予測値の絶対値の2乗を平均したものです。 値が大きいほど誤差の多いモデルと言えます。計算式は以下となります。

    この式は、最小二乗法に用いた誤差を平均化したものです。 小さいほど、もっともらしい直線というのは直感に反しません。 ただし、目的変数の分散が大きい場合は、MSEも大きくなります。

    MSEを求める関数は、sklearn.metrics.mean_squared_errorです。 以下のように、インポートして用います。

    from sklearn.metrics import mean_squared_error 

mse = mean_squared_error(y, y_pred) 
print('MSE:', mse)
    実行結果 
    MSE: 43.60055177116956

    決定係数#

    決定係数 (R2, R-squared, coefficient of determination) は、 MSEを使って、回帰モデルの当てはまりの良さ を示す指標です。寄与率とも呼ばれます。

    最も当てはまりの良い場合、つまり誤差のない場合、1.0 となります。 (当てはまりの悪い場合、マイナスとなることもあります)。 MSEと異なり、目的変数の分散に依存しないため、目的変数のスケールが異なっても一般的な指標となります。

    決定係数を求める
    from sklearn.metrics import r2_score

r2 = r2_score(y, y_pred)
print('決定係数(R2):', r2)
    実行結果 
    決定係数(R2): 0.48352545599133423

    仮説と検証#

    データサイエンスの目的のひとつは、 仮説を立ててより精度の高い予測モデルを作成することです。

    ボストン住宅価格データセットには、部屋数や犯罪率など色々なデータ列があります。 説明変数を選ぶとき、仮説を立てて検証しながら分析するようにしましょう。

    重回帰分析は、二つ以上の説明変数を選びます。

    Let's try

    住宅価格に影響を与えそうなデータ列（説明変数）をふたつ選んで、 回帰モデルを作成し、決定係数を求めてみよう。

    例

    • 部屋数が増えると価格上がりそうだな..
    • 犯罪率が高いと住宅価格は下がりそうだな..
    • 環境汚染は住民にはわかりにくいので、住宅価格に影響しないだろうなど

    このような仮説を立てて、仮説検証をするのはデータ分析の醍醐味です。

    もちろん、不動産のプロフェッショナルではありませんから、何がどう影響するかわかりません。 しかし、回帰モデルに基づいて精度を定量化することで、データとエビデンスに基づいて 議論することが可能になります。

    とりあえず、僕の経験から '通勤'と'教員'辺りが影響を与えるのではないかと睨んで、 説明変数に選んでみました。

    みなさんが、選んだ説明変数に変えて試してみてください。

    X = df[['通勤', '教員']]  # 説明変数
y = df['住宅価格'] # 目的変数
pd.concat([X,y], axis=1).head() #最初の5件のみみてみる

    データ分析と可視化を駆使して、住宅価格に影響を与える説明変数を分析していきます。

    着眼点の一つは相関#

    説明変数の候補を選ぶとき、着眼点の候補となるのが目的変数との相関です。

    ここでは、色々なグラフを紹介しながら仮説を立てるヒントを紹介します。

    仮説:「犯罪率が高ければ住宅価格が低くなりそうだな」

    jointplot
    sns.jointplot(x="犯罪率",y="住宅価格",data=df)
plt.show()

    仮説(2) 工場（NOx は住宅価格に影響ない (カーネル図)

    カーネル図
    sns.kdeplot(data=df, x="工場", y="住宅価格", shade=True, thresh=0.05)
corr = pearsonr(df['工場'], df['住宅価格'])

    cor=df.corr()
plt.figure(figsize=(20,10))
sns.heatmap(cor[(cor >= 0.4) | (cor <= -0.4)], annot=True, cmap='Blues')

fig.subplots_adjust(hspace=0.5) # 間隔を調整
for i, ax in enumerate(fig.axes):

    練習問題#

    ボストン住宅価格データセットを用いて、'住宅価格'を目的変数として、回帰分析してみよう。

    1. 単回帰分析で、決定係数(R2)が最大になる説明変数を探してみよう。
    2. 重回帰分析で、決定係数(R2)が最大になる２つの説明変数の組み合わせを探してみよう。
    3. 自分なりに仮説を立てて３つの説明変数を選び、 重回帰モデルの決定係数(R2)から選んだ説明変数が住宅価格の特徴をよく説明しているか考察してみよう。
    4. （予習）全説明変数の重回帰モデルと決定係数を求める

    考察をするときは、相関係数を意識すると良いでしょう。 特に、重回帰モデルでは、説明変数の組み合わせによっては、 多重共線性(multi-colinearlity、和製英語: マルチコ) が生じることがあり、 相関が高い説明変数を選ぶだけではうまくいかないこともあります。 多重共線性（マルチコ）を調べて、考察に加えてみるのもいいでしょう。

    マルチコへの対処

    マルチコは、統計学的には放置していけないですが、 機械学習的には、モデルの説明力を低下させるものの、予測精度にあまり影響を与えないと言われています。

    また、説明変数を全て入れると、一見、最もよい回帰モデルが作れそうな気がします。 次回の予習として、全説明変数の重回帰モデルも作って、考察に加えてみてください。
