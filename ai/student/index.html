    コースワーク１： データの傾向をつかもう#

    コースワークは、データサイエンスや機械学習をしっかり学ぶための応用問題です。 応用問題なので、今までの内容を振り返るだけでなく、 より深く知るためにWebで調べながら取り組んでみてください。

    今回は、データ分析、グラフ描画、相関などを使いながら、 データの傾向をつかんでいきます。

    データ#

    今回は、ポルトガルの高校生の数学の成績データを使って、 データ解析をします。

    データのURL
    https://kuramitsulab.github.io/data/student-mat.csv

    まず、データをダウンロードして、pandas のデータフレームに読み込んでください。

    Warning

    必要なモジュールのインポートを忘れないにしましょう

    カラム名に続いて、データの個数、さらに形式がint64かobjectのように示されています。

    • int64: 数値データ
    • object: 文字列データ

    データの形式で重要なのは、 数値データ（量的データ）、カテゴリデータ（質的データ）を区別することです。

    • 数値データ: 量が数値によって連続的に表現されるデータであり比較可能。（例. 人数や金額など)
    • カテゴリデータ: カテゴリやグループを表す不連続のデータ

    少し注意したいのは、カテゴリデータであっても、カテゴリの種類が数値で表現されていると、int64形式となります。 また、カテゴリーデータであっても、比較可能な数値に変換できるものがあります。

    必ずしもデータの形式を変換する必要はありませんが、 カテゴリデータとして処理したいなら、型をobjectに変更しておくとよいでしょう。

    数値を文字列に変換する
    df['Medu'] = df['Medu'].astype('object')
df['Fedu'] = df['Fedu'].astype('object')

    データをわかりやすくする#

    海外で作られたデータは、特に英語をベースにした記述であって、直感的なデータ分析の妨げになります。 カラム名やカテゴリーデータは、もう少し直感的に理解しやすいように書き換えてしまいましょう。

    カラム名は、変更後のカラム名の一覧を作成すれば、まとめて差し替えられます。

    カラム名の変更
    columns = [
    '学校名', '性別', '年齢', '住所', '兄弟姉妹', '住居',
    '母学歴', '父学歴', '母職業', '父職業',
    '進学理由', '保護者', '通学時間', '学習時間',
    '落第数', '奨学金', '親もち', '塾', '部活', 
    '幼稚園', '大学', 'ネット', '恋人', '家族', 
    '自由時間', '外出', 'アルコール', 'パリピ',
    '健康状態', '欠席日', 'G1', 'G2', 'G3'
]
df.columns = columns  #差し替え

    カテゴリーデータも新旧対応表を辞書で作れば、まとめてマッピングできます。

    カテゴリデータの置き換え
    df['学校名'] = df['学校名'].replace({'GP': 'GP高', 'MS': 'MS高'})
df['性別'] = df['性別'].map({'M': '男性', 'F': '女性'})
df['住所'] = df['住所'].map({'R': '郊外', 'U': '市内'})
df['兄弟姉妹'] = df['兄弟姉妹'].map({'LE3': 'なし', 'GT3': 'あり'})
df['住居'] = df['住居'].map({'A': '下宿', 'T': '同居'})
df['母職業'] = df['母職業'].map(
    {'at_home': 'なし', 'services': 'サービス', 'teacher': '教師', 'health': '医療', 'other': 'その他'})
df['父職業'] = df['父職業'].map(
    {'at_home': 'なし', 'services': 'サービス', 'teacher': '教師', 'health': '医療', 'other': 'その他'})
df['進学理由'] = df['進学理由'].map(
    {'course': '教育内容', 'home': '通いやすさ', 'reputation': '評判の良さ', 'other': 'その他'})
df['保護者'] = df['保護者'].map({'mother': '母親', 'father': '父親', 'other': 'その他'})

    カテゴリデータの一覧#

    数値データは、describe()メソッドを使って、要約統計量を求めることができました。 カテゴリデータは、便利なメソッドは用意されていませんが、少し工夫すると、 要約統計量をつかむことができます。

    性別の比率をみる
    df['性別'].value_counts()

    ひとつずつ、カラム名を調べるのは大変なので、 df.dtypesがobjectあるものをカテゴリデータとみなして、 カテゴリデータのカラム名を取り出します。

    カテゴリデータのカラム名一覧
    list(df.dtypes[df.dtypes == 'object'].index)

    あとは、for文を使ってカラム名を順番に処理していきます。

    Pandas力をあげるためには？

    Pandas は高機能で、Google で調べるといってもなかなか求めている機能を探すことが難しいです。 最初のうちは、よくわからないコードが出てきたら、地道にひとつずつ動きを確認するようにしましょう。 

    df.dtypes
df.dtypes == 'object'
df.dtypes[df.dtypes == 'object']
df.dtypes[df.dtypes == 'object'].index
    こういう地道な努力がプログラミング向上に繋がります。

    コースワーク#

    ポルトガルの高校生データをデータ分析し、最終成績G3に影響を与える要因を報告しよう。

    （レポートには、Pandas で用いて作画したグラフをエビデンス（根拠）として含めること）

    データ分析は、自由な視点から自由に分析して良い。

    例えばの例

    1. どの属性（カラム）が最終成績に最も影響を与えるだろうか？
    2. そもそも男子学生と女子学生とどちらが成績がいい？
    3. 恋人の有無と成績の関係は？

    もし自分なりに新しい事実（関係性）を発見したときは、そちらを報告して頂けると 評価は高くなります。また、いろいろなグラフの作画に挑戦してください。

    様々なグラフの作画#

    グラフの作画は、講義で習ったものだけでなく、Web等で調べて新しいグラフに挑戦しよう。 Webで調べて書くのが、データ分析を上達するコツです。

    箱ひげ図#

    箱ひげ図は、最大値、最小値、中央値、四分位範囲を視覚化してくれるグラフです。 ヒゲの上端が最大値、ヒゲの下端が最小値を示します。データの範囲がみえてきます。

    G1,G2,G3と並べる
    plt.boxplot([df['G1'], df['G2'], df['G3']])
plt.grid(True)

    seaborn のboxplot()を用いれば、もう少しこった凝った箱ヒゲ図を描画できます。

    order_by = ['なし', 'サービス', '教師', '医療', 'その他']
sns.boxplot(x=df['父職業'], y=df['G3'], order=order_by)
plt.title('父親の仕事が子供の成績に与える影響')
plt.xlabel('父親の仕事')
plt.ylabel('成績')
plt.show()

    subplot を使うと、複数のグラフを一枚の図に加えることができます。

    subplot を使った凝ったグラフ
    plt.figure(figsize= (15,5))
plt.subplot(1, 2, 1)
order_by=['なし', 'サービス', '教師', '医療', 'その他']
sns.boxplot(x=df['父職業'], y=df['G3'], order=order_by)
plt.xticks(rotation=90)
plt.title('父親の職業と成績')

plt.subplot(1, 2, 2)
sns.boxplot(x=df['母職業'], y=df['G3'], order=order_by)
plt.xticks(rotation=90)
plt.title('母親の職業と成績')

plt.show()

    ヒストグラム#

    おなじみのヒストグラムもSeabornを使うと多彩に描画できます。

    countplot
    sns.set_theme(style="whitegrid", font="IPAexGothic")
sns.countplot(x=df['G3'], palette='gray_r')
plt.xlabel('G3')
plt.ylabel('Count')
plt.show()

    hueを追加すると、カテゴリーごとに色分けされたヒストグラムが得られます。

    性別ごとにヒストグラム
    sns.set(font="IPAexGothic")  # 文字化けしたら
sns.countplot(x='G3',hue='性別', data=df)
plt.title('性別ごとの成績分布')
plt.xlabel('G3')
plt.ylabel('Count')
plt.show()

    カーネル密度推定法を使うと、ヒストグラムを滑らかな曲線に保管してくれます。

    カーネル密度推定法によるヒストグラム
    sns.kdeplot(df.groupby('性別').get_group('男性')['G3'], shade = True, label = '男子学生')
sns.kdeplot(df.groupby('性別').get_group('女性')['G3'], shade = True, label = '女子学生')
plt.xlabel('G3')
plt.ylabel('% data distribution')
plt.show()

    ３つ以上のヒストグラムと散布図を一度に見る#

    Seabornのpairplotを用いると、３つ以上のヒストグラムと散布図を一度に確認できます。

    sns.pairplot(df, hue='性別')

    確認したい項目に絞った方が見やすいグラフになります。

    sns.pairplot(df[['欠席日', 'アルコール', '年齢', 'G3']])

    相関行列とヒートマップ#

    相関行列は、すべての数値データ間で相関係数を計算した行列です。

    相関行列を求める
    df.corr()

    ご多分に漏れず、可視化しておくとデータ解析が行やすいです。

    相関行列のヒートマップ
    plt.figure(figsize=(10,10))
corr = df.corr()
sns.heatmap(corr, annot=True)
