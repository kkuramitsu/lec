    CSVファイルとは

    CSV ファイルは、表計算ソフト(Excel)のファイルをプログラムからデータ処理しやすくなるように、 カンマ区切り形式のテキストとして出力したものです。 当然、Excel 等の表計算ソフトでも開いてみることができます。

    今回は、このようなCSVファイルをPythonから読み込んでデータ分析を行う方法を学びます。

    Pandas#

    Pythonを使って表データを扱うときは、pandas と呼ばれるモジュール（ライブラリ）を使います。

    pandasの準備（これは決まり文句）
    import pandas as pd

    これで、Pandas をインポートして、pdという名前から利用できるようになりました。 pdは慣習的に使われる短縮された名前です。 pd.という接頭辞がついた関数は、全てPandas のライブラリ関数となります。

    CSVファイルの読み込み#

    早速、Pandasを使って、先程ダウンロードした'boston.csv'を読み込んでみます。

    ファイル名を指定して、CSVファイルを読み込む
    df = pd.read_csv('boston.csv')

    ここでは、読み込んだデータは、データフレームに頭文字に由来するdfという変数名にしています。 変数名をつけておくことで、後から変数名を通してデータ操作できるようになります。

    データフレーム

    Pandas では、表データのことを データフレーム(data frame) と呼びます。 だから、頭文字をとって dfと変数名をつけることが多いです。

    pd.read_csv(filename)

    filename で指定されたCSVファイルを表データ（データフレーム）として読み込む

    メソッド操作によるデータフレームの確認#

    それでは、データフレーム dfを操作して、先頭のデータを確認しましょう。

    データフレームの先頭を見る
    df.head()

    変数名とドット.に続く、head()は、 データフレーム df への操作の種類を表し、メソッドと呼ばれます。

    • df.head(10): メソッド操作においてパラメータを渡す
    • df2.head(): 別のデータフレームdf2へのメソッド操作

    無事に、表データの先頭行が見ることができたでしょうか？ 次に紹介するhead以外のメソッドも試してみてください。

    データフレームの確認

    Pandas では、データフレームの操作を行います。そのとき、その内容を確認します。 次のメソッドも、データフレームの確認によく用いられます。 試してみましょう。

    コード 例
    df.head(10) dfの先頭の10行を得る
    df.tail() dfの末尾を得る
    df.sample(10) dfからランダムに10行取り出す

    Python とエラー対応#

    Pythonなどプログラミング言語は、自然言語とは異なり、厳密な文法と解釈が決まっています。 少しでも書き方のルールから外れると、エラーになって動作しません 。

    ここでは、Python初心者がよくつまずくエラーを見ながら、エラー対応をみていきましょう。

    よくあるエラーの例

    データ分析の第一歩#

    データ分析は、各カラム（データ列）を理解する ところから始めます。

    もう一度、ボストン住宅データセットのデータフレームを見てみましょう。

    今度はデータの中身に着目
    df.head()

    ボストン住宅価格データセットには、次のようなデータ列が含まれています。

    • crim '犯罪': 人口1人当たりの犯罪発生数
    • zn '自然': 25000平方フィート以上の居住区間の占める割合
    • indus '工場': 小売業以外の産業が占める面積の割合
    • chas '川近': チャールズ川によるダミー変数（1：川の周辺，0：それ以外）
    • nox '大気汚染': NOxの濃度
    • rm '部屋数': 住居の平均部屋数
    • age '築年': 1940年より前に建てられた物件の割合
    • dis '通勤': ボストンの主な職場までの距離
    • rad '交通': 幹線道路へのアクセスのしやすさ
    • tax '資産税': ＄10,000ドルあたりの不動産税率の総計
    • ptratio '教員比': 児童と教師の比率
    • black '黒人': アフリカ系アメリカ人居住者の割合
    • lstat '低所得': 給与の低い職業に従事する人口の割合
    • medv '住宅価格': 住宅価格 (所有者が占有している家屋の＄1000単位の中央値)

    オリジナルのボストン住宅価格データセットは、crim のように英語で列名が与えられています。 ただし、本講義では、より直感的にデータ分析しやすいように'犯罪'のように和訳に変えてあります。 ただし、少し意訳してあるので、オリジナルなデータの意図と少し違うかもしれません。

    データの公平性と修正#

    データサイエンスやAIはデータを扱います。 データは過去の事実であっても、それが妥当なものとは限りません。 例えば、女性の国会議員比率が低いからと言って、 女性の能力が低いと分析するのは間違っています。 不当な差別や社会的な抑圧によるものかも知れません。

    人工知能の公平性（Fairness)

    不公平なバイアスに基づき学習した機械学習モデルが不当な差別（人種差別／民族差別や、性別差別、文化差別／地域差別など）を 引き起さないように不公平なバイアスを排除すること

    ボストン住宅価格データセットは、もともと1970年代のデータです。 現在の常識から考えると相応しくないデータが含まれています。 '黒人'などのデータ列は、例え、黒人比率が住宅価格に影響を与えるものであっても、 相応しいものとは言えません。

    このような差別につながるデータは、削除するようにしましょう。

    Pandasでは、dropメソッドを使うと、指定したデータを削除できます。

    `黒人`のデータ列を削除する
    df.drop('黒人', axis=1, inplace=True)
df.head()

    データ列と種類⭐️⭐️#

    Python 経験者向けの内容

    Python初心者は、このセクションを飛ばして構いません

    Python を使いこなすためには、どのようなデータ型で値が得られているか知ることが大切です。 なぜなら、データ型によって、操作できるメソッドの種類が異なるからです。

    データ列は、Series と呼ばれる pandas のデータ型で得られます。

    df['住宅価格']の型を調べる

    要約統計量: データの特徴を掴む#

    数値データの特徴は、統計的なデータの散らばり で記述されます。 データ列を取り出して眺めているだけでは、データの特徴を掴むことができません。

    describe()

    データフレームやデータ列に対して、基本的な統計量を求める便利なメソッドです。 基本的な統計量のことを 要約統計量 とも言います。

    データ列'住宅価格'の要約統計量を得る 

    df['住宅価格'].describe()
    実行結果
    count    506.000000
mean      22.532806
std        9.197104
min        5.000000
25%       17.025000
50%       21.200000
75%       25.000000
max       50.000000

    各項目の意味

    • count: データ数
    • mean: 平均値
    • std: 標準偏差
    • min: 最小値
    • 25%: 下位25%目の値（第一四分位点）
    • 50%: 中央値
    • 75%: 上位25%目の値（第三四分位点）
    • max: 最大値

    正規分布#

    統計的にデータ値の散らばりを見るときは、 正規分布が大切になります。

    • （正規分布を仮定すると） データの散らばりは、平均と標準偏差で分布を記述できます。つまり、データ値の90%は、平均値から標準偏差の範囲に分布していると見なせます。
    • （正規分布から大きく外れると） 分布の非対称性や外れ値から影響を受けにくい中央値、四分位点、最大値・最小値などの値が参考になります。

    正規分布からのずれを知るためには、尖度や歪度などの高次モーメントから求められる統計量があります。

    歪度(Skewness)

    分布の非対称性を示す指標。値が正であると分布が右に伸びている。逆に負であると左に伸びている。

    尖度(Kurtosis)

    正規分布を基準とした分布の鋭さの指標。基準である正規分布の尖度よりも大きければ分布は尖っており、小さければ緩やかな分布である。

    Let's try: 個々の統計量を得る

    describe() メソッドは、要約統計量をまとめて出力してくれましたが、 個々の統計量を求めるメソッドもあります。試してみましょう。

    メソッド 例
    df['A'].count() 'A'列のデータ件数
    df['A'].mean() 'A'列の平均値
    df['A'].std() 'A'列の標準偏差
    df['A'].var() 'A'列の分散
    df['A'].max() 'A'列の最大値
    df['A'].min() 'A'列の最小値
    df['A'].median() 'A'列の中央値
    df['A'].mode() 'A'列の最頻値
    df['A'].skew() 'A'列の歪度
    df['A'].kurt() 'A'列の(Fisherの定義による)尖度

    データの可視化#

    データ列の特徴は、数列や要約統計量を見ていても、直感的に傾向をつかめません。 一方、次の図は、ボストン住宅価格のヒストグラムと正規分布を仮定した近似曲線を可視化してみた例です。 このようにグラフに可視化することで、よりデータは理解しやすくなります。

    Python で グラフを描画するライブラリ（モジュール）は、次のようなものがあります。

    • matplotlib: 定番（というか必須）
    • seaborn: より綺麗なグラフが書ける

    これらのモジュールを使えるように、グラフを描画する前には必ずインポートしておきます。

    matplotlib,seabornの準備 

    import matplotlib.pyplot as plt
import seaborn as sns
try:
    import japanize_matplotlib #matplotlibの日本語化

    os.system('pip3 install japanize_matplotlib')
    import japanize_matplotlib 
sns.set(font="IPAexGothic") #日本語フォント設定

    Matplotlibやseabornは、海外で開発されたライブラリなので、日本語表示が文字化けします。 グラフ上に日本語を表示したいときは、japanize_matplotlibをインストールして用いることになります。

    箱ひげ図#

    まず、箱ひげ図を書いてみましょう。 最近は小学生のときに習うらしいので、皆さんおなじみでしょう。

    Pandas で箱ひげ図を描画するときは、plt.boxplot()に描画したいデータ列を引数で与えます。

    `'住宅価格'`の箱ひげ図
    plt.boxplot(df['住宅価格'])
plt.show()

    グラフの見方

    • 四角の部分は、四分位範囲を表し、箱 と呼ばれます。
    • 箱の上線が上位25%、箱の下線が下位25%、オレンジの横線が中央値を表します。（データの半分がこの範囲にあることになります。）
    • 四分位範囲の1.5倍以上、離れた点は 外れ値 とみなし、○でプロットされます。
    • 外れ値でない最も外側の点が ひげ となります。(ひげは、厳密には最大値、最小値を表すわけではありません。)

    外れ値の発生頻度は？

    正規分布である場合、箱ひげ図における外れ値(1.5倍)とは0.8%の確率でしか発生しない稀なデータとなります。

    plt.boxplot(データ列)

    データ列を指定すると、その箱ひげ図を描画する

    ヒストグラム#

    ヒストグラムは、縦軸に度数、横軸に階級をとり、 データの分布状況を視覚的に認識するための定番グラフです。

    Pandas でヒストグラムを描画するときは、plt.hist()に描画したいデータ列を引数で与えます。

    '住宅価格'列のヒストグラム

    plt.hist(df['住宅価格'])
plt.show()

    plt.hist(x)では、オプションを指定すると、描画を調整することができまう。ビン（表示する棒の数、階級数）は、デフォルト値（何も指定しない場合の初期値）では10になっていますが、bins=20のように変更することができます。

    オプションでビンを指定 

    plt.hist(df['住宅価格'], bins=20)
plt.show()

    Matplotlibやseabornは、さまざまなオプションが用意されています。 Google などで、グラフの書き方をどんどん調べて、自分の求めているグラフを書けるようにしましょう。

    演習問題#

    ボストン住宅データセットから'住宅価格'以外の３つのデータ列（カラム）を選んでみよう。 （何を選んでも構いません。）

    1. 選んだデータ列の最大値、最小値、平均値を求めてみる
    2. 選んだデータ列の標準偏差、分散を求めてみる
    3. 選んだデータ列の箱ヒゲ図を描画してみる
    4. 選んだデータ列のヒストグラムを描画してみる
    5. 選んだデータ列が正規分布にしたがうか従わないか、従わないか考えてみよう

    (おまけ） 正規分布の近似曲線⭐️⭐️#

    さて、演習問題の考察を深めたい学生のため、 可視化の冒頭で紹介した正規分布の近似曲線とヒストグラムを合わせる方法を見てみましょう。

    正規分布の近似曲線を描画するためには、 確率密度関数(probability density function)を使います。

    この関数は、SciPy モジュールの scipy.stats.norm.pdf()にあります。 plt.plot(x, y)は、線グラフを描画する関数です。

    import numpy as np
from scipy.stats import norm

x = np.linspace(-5, 5, 100)  # 区間[-5,5]を100等分する
y = norm.pdf(x)  # 確率密度関数
plt.plot(x, y)
plt.show()

    データ列'住宅価格'の最大値、最小値、平均値、標準偏差からスケールを調整します。 そして、ヒストグラムを組み合わせると: 

    bins = 20
k = (df['住宅価格'].max() - df['住宅価格'].min()) / bins  # 区間幅
x = np.linspace(df['住宅価格'].min(), df['住宅価格'].max(), 100)
y = norm.pdf(x, df['住宅価格'].mean(), df['住宅価格'].std())* len(df['住宅価格']) * k

plt.hist(df['住宅価格'], bins=bins)  # ヒストグラム
plt.plot(x, y, linestyle='--') # 確率密度関数
plt.show()

    （おまけ２） シャピロ・ウィルク検定(Shapiro-Wilk test)⭐️⭐️#

    シャピロ・ウィルク検定(Shapiro-Wilk test)は、 「標本は、正規分布に従う母集団から無作為抽出されたものである」を帰無仮説として、 正規分布を判定する方法です。 特に、データ数が1000以下の有効とされます。

    Python では、SciPyモジュールのshapiro関数で算出できます。

    '住宅価格'列のシャピロ・ウィルク検定
    from scipy.stats import shapiro

print(shapiro(df['住宅価格']))

    (0.9171748757362366, 4.940195786292171e-16)

    検定統計量とp値が得られます。ここでは、p値が有意水準(0.05)より小さいかどうか見ます。 `df['住宅価格'])は、0.05より小さいので、「正規分布である」という仮説が棄却されます。 つまり、「正規表現と言えない」母集団となります。
